Changelog

Version history for Scrobble Time Machine

← Back to app

v2.0

2025-03-05 Major
  • On This Day — new mode that plays what you were listening to on today's date in a random past year
  • Artist autocomplete — suggestions from your top 500 artists as you type, with play counts
  • Remember username — your Last.fm username is saved to localStorage
  • Keyboard shortcuts — Escape to cancel, Enter to go
  • Smarter artist search — two-phase find-then-zoom algorithm finds dense listening sessions

v1.9

2025-03-03 Feature
  • Multi-file architecture — split from single HTML file into separate JS, CSS, and HTML files
  • Favicon — red rewind icon in browser tab

v1.8

2025-03-03 Feature
  • Mood search — pick a vibe (Chill, Melancholy, Energetic, Raw, Dreamy, Soul, Indie) and find matching tracks using Last.fm tags
  • Dynamic year dropdown — date picker starts from your actual first scrobble year

v1.7

2025-03-03 Feature
  • Save as Playlist — save matched tracks as a private Spotify playlist
  • Barlow font — updated typography to match Last.fm's style

v1.6

2025-03-03 Fix
  • Session tracking — detects when another device takes over Spotify playback
  • Multi-device safety — click any track to reclaim your session

v1.5

2025-03-03 Fix
  • Auto-continuation bugfix — fixed foundCurrent guard preventing tracks from loading

v1.4

2025-03-02 Feature
  • Date picker — jump to a specific year, month, or day
  • Artist search — find pages where a specific artist appears

v1.3

2025-03-02 Feature
  • Now-playing tracking — highlights the currently playing track in real time
  • Auto-continuation — next batch loads automatically as you listen

v1.2

2025-03-02 Feature
  • Smart matching — deduplicates searches, caches results
  • Rate limit strategy — respects Spotify API limits with batching and delays

v1.1

2025-03-01 Feature
  • Interactive timeline — drag the slider to scrub through your entire listening history
  • Mobile fixes — responsive layout improvements
